HTML이란?

HTML이란 하이퍼 텍스트 마크업 언어(영어: Hyper Text Markup Language, HTML)로, 웹 브라우저에 표시되도록 설계된 표준 마크업 언어다.

생활코딩
  1. tag 문법
    • tag는 HTML의 문법
    • br 태그 : 단락을 구분할 때 사용하는 줄바꿈 태그. / 안 닫아도 됨.
    • p 태그 : 단락을 구분하는 태그이며 / 닫아줌 필요. 문서의 단락임을 표현하는 기능이기 때문에 br보다는 정보로서 더 가치있는 HTML임. css를 함께 이용해서 원하는 간격을 만들 수 있음.
    • 여러 태그는 (부모태그, 자식태그) 관계가 계속 바뀜. 부모-자식 한상 연결된 태그 종류가 있음.
    • li 태그: 목차를 나눌때는 li 태그를 쓰고 그 앞에 부모 태그인 ul (unorderd list) 혹은 ol (ordered list)태그를 써서 여러 li 태그 구분 가능
    • title 태그 : 웹 제목 표시
    • 검색엔진에 노출되기 위해서는 웹의 '정보'를 탄탄히 하는게 중요함. Accesibility (정보의 접근성, 개방성)
    • a 태그 : 링크 삽입 가능
  2. 이미지 속성 및 css 스타일 설정
    • 이미지 넣기 : img 태그 + src 속성(Attribute)에 이미지 주소 넣기. Tip) unsplash.com에서 저작권 없는 이미지 참조 가능
  3. 필수 구조 및 특성
    • 컴퓨터는 모든 정보를 최종적으로 0 또는 1로 저장함. 0과 1을 어떻게 저장할지에 대한 약속 중 하나가 'utl-8'임.
    • html파일을 웹브라우저에서 열때 utf-8로 열리게끔 저장할 수 있음. meta태그 charset="utf-8"로 코딩.
    • title은 head태그로 묶고, 본문은 body태그로 묶는게 정해진 HTML 구조임
    • 관용적 표현으로 !doctype html 태그로 가장 상단에 적어야 함. 그 아래 html 태그 적어야 함. 모든 웹은 이 구조로 만들어짐.
  4. Tip
    • html 사용 설명서 (w3c 국제 기구)에서 여러 태그 검색해보면 됨.
